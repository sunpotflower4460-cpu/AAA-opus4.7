# Phase 16 UI/UX Integrity Polish Report
## 目的
Phase 11〜15で追加したUI/UX要素を総合監査し、過剰演出・説明過多・視覚的なズレを削ぎ落とした。

---

## 1. 監査対象
- ホーム画面: 上部情報量、紙片コピー、検索位置、月の干渉を確認
- 空状態: 文言量、CTA密度、装飾の強さを確認
- 紙片コピー: 中央重心、英語補助の必要性、余白を確認
- 紙材質: 背景・紙面・カードの質感強度と可読性を確認
- メモカード: 日付印、タイトル、本文プレビューの階層を確認
- エディタ: 上部構成、本文優先度、保存表示の静けさを確認
- 日付印: サイズ、濃さ、アニメーション強度を確認
- 保存アニメーション: duration、残り方、reduced motion対応を確認
- 月モチーフ: サイズ、明るさ、位置、紙目との干渉を確認
- 広告: 一覧下部配置、カード誤認防止、Premium時非表示を確認
- Premium: 文言密度、一覧下部配置、購入復元導線を確認
- レスポンシブ: 375 / 390 / 430 / 768 / 1024px を headless Chrome で確認

---

## 2. 見つかった問題
- ホーム上部に説明ブロックと英語補助が重なり、主コピーより説明が残りやすかった
- 空状態の呼吸アニメーションとCTAがやや強く、最初の静けさを削っていた
- 紙材質・月・日付印・保存表示が同時に主張し、エディタの本文より先に視線を引く瞬間があった
- メモカードの補助ラベル、広告英語文、Premium英語文が一覧下部の密度を上げていた

---

## 3. 削ぎ落とした内容
- ホームの追加説明ブロックと紙片英語サブコピーを外した
- 空状態の呼吸アニメーションと検索空状態の英語補助を外した
- メモカードの「最後の余韻」ラベルと保存表示の英語補助を外した
- 紙目、月光、カード影、日付印の発光、保存演出の強度を下げた
- 一覧下部の広告 / Premium カードから英語補助文と重複見出しを外した

---

## 4. 修正した内容
### ホーム
- 上部構成を「残心 / ZANSHIN NOTES / 紙片コピー」に整理した
- 検索バーと一覧幅を `max-w-[430px]` に揃え、PCでも静かな密度に寄せた

### 空状態
- 推奨文言に沿ってコピーを短くした
- CTAサイズと装飾を少し抑え、最初の一歩だけを残した

### メモカード
- 本文プレビュー前の補助ラベルを削除した
- プレビュー文字数とお気に入り印の主張を少し下げた

### エディタ
- 日付印を紙面内の上部へ移し、本文へ流れる構成に整理した
- タイトル / 本文 placeholder を静かな文言へ差し替えた
- 保存表示は日本語のみの短い反応に絞った

### 日付印
- 文字サイズ、色、月印、発光レイヤーを淡くした
- 定着アニメーションを 900ms → 640ms 相当へ短縮した

### 月モチーフ
- 月のサイズ、opacity、blur、発光を抑えて背景奥へ沈めた
- iPhone幅とPC幅で右上の逃がし方を再調整した

### 広告/Premium
- 一覧下部だけに留まることを維持した
- 広告カードとPremiumカードの表示文言を短くし、売り込み感を弱めた

## 5. 残心らしさの確認
### 書く体験は静かか？
OK
理由:
- エディタ紙面を軽くし、日付印を紙面上部へ収め、本文入力欄の高さを広げた
- 保存表示は短く淡く残るだけにし、広告 / Premium はエディタへ出ないことを確認した

### 説明しすぎていないか？
OK
理由:
- ホーム説明ブロック、空状態の過剰補助、一覧下部の英語補助を削った
- 主要画面で残る説明は主コピーと最小限の導線だけになった

### 紙材質は過剰ではないか？
OK
理由:
- 背景ノイズ、紙目、カード影、紙片影を一段落とし、可読性優先へ戻した
- 古紙感は残しつつ、古地図や羊皮紙に寄りすぎないようにした

### 月は主張しすぎていないか？
OK
理由:
- 月のサイズと発光を下げ、右上の画面外へ深く逃がした
- 文字面と重なりすぎない位置で、時間の気配だけを残す状態に寄せた

### 日付印は自然か？
OK
理由:
- 日付印は小さく淡くし、タイトルより前に出ない階層に戻した
- 定着アニメーションも短くし、保存通知ではなく紙への定着に寄せた

### 広告/Premiumは邪魔していないか？
OK
理由:
- 空状態とエディタでは非表示、一覧下部のみであることを確認した
- Premium購入後は広告が消え、購入復元導線も一覧下部に維持される

## 6. 画面幅確認
- 375px: 紙片は中央維持。検索・カード・エディタ紙面は 333px 幅で収まり、空状態に広告 / Premium なし
- 390px: iPhone基準幅でも上部コピーと検索が詰まらず、本文領域は 304px 幅を確保
- 430px: 広めのiPhone幅でも月は右上奥に留まり、本文領域は 344px 幅を確保
- 768px: 検索 / 一覧は 430px に抑え、エディタ紙面は 560px 幅で中央配置
- 1024px: ホームと一覧は静かな中央密度を維持し、エディタは 560px 幅で過度に広がらない
