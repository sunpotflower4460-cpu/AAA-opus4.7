# 残心 — デザインシステム / Design System

## デザイン原則

| 原則 | 意味 |
|------|------|
| **Calm** | 落ち着いた、静かな雰囲気 |
| **Minimal** | 必要なものだけを残す |
| **Spacious** | 呼吸できる余白 |
| **Japanese-inspired** | 和の美意識をベースに |
| **iOS-first** | iPhoneの手のひらに収まるUI |
| **Golden ratio aware** | 黄金比を意識したスケール |
| **Quiet but memorable** | 静かだが、印象に残る |

---

## 黄金比スケール

余白・サイズ・行間には、可能な範囲で以下のスケールを使う。

```
4 / 8 / 13 / 21 / 34 / 55 / 89
```

### 使用例

| 用途 | 値 |
|------|----|
| 画面左右余白 | 21px |
| カード padding | 21px |
| セクション間隔 | 34px |
| FABサイズ | 55px |
| エディタ上部余白 | 34px |
| 本文行間 | 1.618前後 |

## タイポグラフィ

### 見出し（静かな明朝寄り）

```css
font-family:
  "Hiragino Mincho ProN",
  "Yu Mincho",
  "Noto Serif JP",
  "Noto Serif",
  serif;
```

### 本文入力（読みやすさ優先）

```css
font-family:
  "Hiragino Sans",
  "Noto Sans JP",

  sans-serif;
```

## モチーフの使い方

### 刀

**意味：** 集中・研ぎ澄ます・不要なものを削る・決断

**UI利用例：**
- 細い区切り線（`1px solid` / Faded Ink）
- 新規作成ボタンの抽象的アイコン
- メモカード左の縦アクセントライン

---

### 扇

**意味：** ひらく・広がる・呼吸・優雅さ

**UI利用例：**
- 空状態（Empty State）の薄い背景モチーフ
- お気に入りアイコンの造形ヒント
- 展開アニメーションの参考

---

### 円相

**意味：** 未完成の美・循環・余韻・静かな集中

**UI利用例：**
- 保存完了時の微細なアニメーション（小さな円が静かに描かれる）
- ロゴや透かし表現
- ローディング表現（派手にしない）

---

## アニメーション方針

### 使用してよい

- `fade` — ゆっくりフェードイン/アウト
- `slow slide` — ゆっくりスライド
- `soft scale` — わずかな拡縮
- `breath-like motion` — 呼吸するような動き

### 避ける

- 派手なバウンド
- 意味のないキラキラ・パーティクル
- ゲーム的な演出
- 速すぎるトランジション（200ms以下は原則使わない）

### トランジション時間の目安

| 種別 | 時間 |
|------|------|
| 画面遷移 | 300–400ms |
| カード表示 | 200–300ms |
| 保存完了フィードバック | 400–600ms |
| ローディング | 呼吸感のあるループ |

---

## コンポーネント方針

- ボタンは少なく、役割を明確に
- アイコンは補助として使い、テキストを代替しない
- カードはシンプルな矩形ベース（過剰な装飾なし）
- 影（`box-shadow`）は控えめ、または使わない
- 角丸は小さめ（`4–8px`）または直角に近い形
