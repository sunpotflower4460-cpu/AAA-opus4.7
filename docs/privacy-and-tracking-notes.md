# Privacy and Tracking Notes

## 現在のデータ方針
- アカウントなし
- ログインなし
- メモはローカル保存
- クラウド同期なし
- AI送信なし

## 収集する可能性のあるデータ
現時点:
- メモ内容: 端末内保存のみ
- アカウント情報: なし
- 位置情報: なし
- 連絡先: なし
- 写真: なし
- ヘルス情報: なし

補足:
- 現時点では広告 SDK 未導入、分析 SDK 未導入
- Premium mock は `localStorage` に保存される仮状態であり、決済情報は扱っていない

- SDK名:
- 収集データ:
- 識別子:
- 使用状況データ:
- 診断情報:

導入前チェック:
- SDK の Privacy Manifest を確認する
- App Store Connect の App Privacy に反映する
- トラッキング広告なら ATT 許諾フローを用意する
- 非パーソナライズド広告で代替できないか検討する

## ATT方針
- トラッキング広告を使う場合は ATT 対応が必要
- できる限り非パーソナライズド広告を優先検討する
- ユーザーに追跡許可を強制しない
- 許可しないとメモ機能が使えない設計にしない

## App Privacy整理メモ
- アプリ本体だけでなく、第三者 SDK の収集データも申告対象として扱う
- App Store Connect では、データ種別・利用目的・トラッキング有無を SDK 単位で確認する
- 広告 SDK や分析 SDK を追加したら、提出前に再棚卸しする

## Privacy Policyに書くべき内容
- メモは端末内に保存される
- 広告SDKを使う場合、そのSDKのデータ収集について説明する
- Premium購入にはApp Storeの決済を使う
- 問い合わせ先
- トラッキングを使う場合の許諾と拒否手段
