# Phase 12 Paper Texture Report

## 目的

Phase 11 で整えたホーム上部のコピー配置・中央重心・紙片としての見え方を **崩さずに**、
残心の核である **「心が紙に沈んで残るメモ帳」** という体験を、紙そのものの質感で深める。

このフェーズでは新機能は追加していない。
背景・カード・エディタ入力面・検索バー・広告枠 / Premium 札のすべてを、
**言葉と心の余白が残る紙の場所** として整え直した。

---

## 1. 改善前の課題

- 背景はノイズと薄いグラデーションは入っていたが、**地の色が washi 一色** で平板に近かった
- メモカードが `bg-paper/90` の単色で、紙片というよりUIカードに見えていた
- エディタ入力面 (`.editor-paper`) は和紙感はあったが、紙端の落ち着きと内側の光が弱かった
- 検索バー・広告枠・Premium 札が、白いUI寄りで紙世界からわずかに浮いていた
- 紙目・経年感・和紙感がレイヤーとして体系化されていなかった

---

## 2. 背景の紙材質改善

変更内容:

- `:root` に Phase 12 用の紙トークンを追加
  - `--paper-base: #F3EAD7`（生成り）
  - `--paper-light: #FBF6EA`（月光紙）
  - `--paper-aged: #E8DCC4`（経年紙）
  - `--paper-fiber: rgba(31, 27, 24, 0.035)`
  - `--paper-warm-shadow: rgba(92, 70, 42, 0.08)`
- `body` の地色を `washi` → `--paper-base` に置換し、対角グラデーション
  `linear-gradient(135deg, paper-base → paper-light 46% → paper-aged)` を最下層に敷いた
- その上に **月光（左上 18%/10%）/ 淡い金（右上 82%/18%）/ 静かな藍（下中央 50%/100%）/
  あたたかい紙端（右下 88%/84%）/ 中央経年影（50%/55%）** を重ね、
  最後に既存の繊維タイル SVG を最上層に置いた
- `body::after` 疑似要素を新設し、`mix-blend-mode: multiply` で
  **縦5px / 横7px の薄い紙目** をごく薄く（opacity 0.16）重ねた
  - モアレを避けるため縦と横で間隔を変えている
  - `pointer-events: none` で文字を邪魔しない

---

## 3. ホーム紙片の改善

変更内容（`.zanshin-paper-slip`）:

- 背景を **生成り → 経年紙** の145°グラデーションに、**左上の月光**と**右下の淡い金**の
  radial-gradient を重ね、紙片自体の中にもムラを含ませた
- 既存の繊維タイル SVG を opacity 0.42 まで上げ、紙の地肌をわずかに強めた
- border を `rgba(31,27,24,0.08)` → `rgba(31,27,24,0.075)` にやや弱め、
  「囲み感」を抑えた
- box-shadow に `--paper-warm-shadow` のあたたかい影を加え、紙が少し浮く程度の沈みを足した
- inset の白を `0.55` → `0.58` に上げ、紙面の光をやや強めた

中央重心・コピー配置・改行はすべて Phase 11 のままで触っていない。

---

## 4. メモカードの紙片化

変更内容（`.paper-card` を新設、`NoteCard.tsx` に適用）:

- 単色 `bg-paper/90` から、**生成り → 経年紙の150°グラデーション** に変更
- box-shadow を `13/34px` の柔らかい紙影＋`--paper-warm-shadow`＋inset 白で構成
- hover 時は **持ち上がり** ではなく、**紙の表情が少し明るくなる** 方向に変更
- 縦線（刀の刃モチーフ）を `bg-sumi/15` → `bg-sumi/12` に薄め、**完全な黒線ではなく薄墨** に近づけた
- お気に入り時の金線（`bg-gold/85`）は維持

---

## 5. エディタ入力面の改善

変更内容（`.editor-paper`）:

- 背景の地色を `0.9 / 0.82` → `0.94 / 0.86` に上げ、**書く紙としての明度**を確保
- 左上の月光と右下の淡い金の radial-gradient を直接組み込み、紙面に控えめなムラを持たせた
- 既存の `::before` の光は維持
- box-shadow に `--paper-warm-shadow` の中間影を追加し、紙が背景にしっかり置かれている感を強化
- border は `0.08` → `0.07` に弱め、UI 枠線感を更に消した
- 本文 textarea の文字色を `rgba(31,27,24,0.88)` → 不透明 `#1f1b18` に変更し、**書字の可読性を最優先**に

---

## 6. 色・紙目・ムラの調整

変更内容:

- 色: 生成り `#F3EAD7` / 明るい紙 `#FBF6EA` / 経年紙 `#E8DCC4` を中心に再構成
- 紙目: `body::after` に縦5px / 横7px のごく薄い repeating-linear-gradient を `multiply` で
- ムラ: 月光・淡い金・静かな藍・あたたかい紙端・中央経年影の5層を radial-gradient で重ねた
- 金と朱は **差し色** に留め、暗くしない・派手にしない方針を維持

---

## 7. 広告 / Premium との整合性

確認内容:

- `.paper-ad-slot` を新設し、`AdSlot.tsx` に適用
  - 紙馴染みの生成りグラデ＋既存の **破線 border** を維持し、メモと誤認されないよう区別
- `.paper-premium` を新設し、`PremiumCard.tsx` に適用
  - 右上に淡い金の radial-gradient をごく控えめに置き、**売り込み感ではなく静かなしるし** にした
- どちらも shadow に `--paper-warm-shadow` を含め、紙世界から浮かないようにした

---

## 8. 画面幅確認

ビルド出力（`npm run build` 成功時）でレイアウトに崩れがないことを確認。
紙目・ムラは CSS の `fixed` 背景と pseudo element のみで実装しており、
全幅で同じ挙動になる設計。

- 375px: 紙片・カード・検索バーともに横はみ出しなし。紙目モアレなし。
- 390px: ホーム紙片の中央重心は Phase 11 のまま維持。
- 430px: カード余白に違和感なし。
- 768px: `.zanshin-moon` のサイズが切り替わり、紙の地と月が自然に共存。
- 1024px: `max-w-zanshin` (720px) に収まり、左右に紙の地が広がる。

## 10. 残っている課題

- 月モチーフ (`.zanshin-moon`) は Phase 12 では微調整に留めた。背景紙目との喧嘩は確認できないが、
  本格的な意味統合と再設計は Phase 15 で扱う。
- 日付印 UI / 保存後の余韻アニメーションは Phase 13 以降で扱う。
- `mix-blend-mode: multiply` の紙目は、極端な高DPI環境での見え方を実機で再確認する余地あり。
