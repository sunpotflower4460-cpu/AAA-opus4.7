# 残心 — コンセプト / Concept

## 残心とは

残心とは、武道や茶道などにある、動作が終わったあとにも意識・気配・余韻が残る状態です。

弓を放ったあとの姿勢。  
茶を点てたあとの静けさ。  
刀を納めたあとの間。

このアプリでは、メモを書くことを単なる入力作業ではなく、  
**心の一部を静かに置く行為**として扱います。

---

## 中核思想

```
書く
　↓
残る
　↓
読み返す
　↓
静まる
　↓
また書く
```

この循環を、アプリ全体で大切にする。

---

## 3つのキーワード

### 1. 残心

- 書いたあとにも気配が残る
- 保存完了を派手にしない
- 自動保存は静かに行う
- 最後の一文や更新時間がやわらかく残る

### 2. 間（ま）

- 詰め込まない
- ボタンを増やしすぎない
- 画面上の情報量を制限する
- 思考が呼吸できる余白を残す

### 3. 余白

- 余白は空白ではなく、思考のための場所
- 背景、カード、文字、行間に余裕を持たせる
- 一画面に情報を出しすぎない

---

## 和の扱い

**使ってよいモチーフ：**

| モチーフ | 意味 |
|----------|------|
| 刀 | 集中・研ぎ澄ます・不要なものを削る |
| 扇 | ひらく・広がる・呼吸・優雅さ |
| 墨 | 深み・静寂・力強い簡潔さ |
| 和紙 | 温かみ・手触り・素朴な美しさ |
| 金箔 | 静かな贅沢・特別な瞬間 |
| 円相 | 未完成の美・循環・余韻 |
| 藍 | 深い集中・日本の色 |
| 朱 | アクセント・大切な印 |

**目指すのは：**

> 古典的な和風アプリではなく、  
> 海外にも伝わる「**静かな現代和**」。

過剰な和風テーマパークにはしないこと。  
モチーフは「意味」として使い、「装飾」として使わない。

---

## このアプリが大切にしないもの

- 機能の多さ
- 派手なアニメーション
- 通知や割り込み
- ソーシャル機能
- 複雑なカスタマイズ

---

## このアプリが大切にするもの

- 書きはじめやすさ
- 読み返したくなる静けさ
- 言葉を置く感覚
- 手のひらの中の余白
- 使うほどに馴染む、変わらないUI
