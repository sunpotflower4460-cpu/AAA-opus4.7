# App Store Monetization Checklist

## IAP
- [ ] Paid Apps Agreement 確認
- [ ] App Store Connect で IAP 作成
- [ ] Product ID 確認
- [ ] StoreKit 実装
- [ ] 購入処理
- [ ] 復元処理
- [ ] Sandbox テスト
- [ ] TestFlight 確認
- [ ] Review 用スクリーンショット
- [ ] Review Notes

## Ads
- [ ] 広告 SDK 選定
- [ ] データ収集内容確認
- [ ] Privacy Manifest 確認
- [ ] App Privacy 更新
- [ ] ATT 必要性確認
- [ ] 非パーソナライズド広告設定検討
- [ ] エディタ非表示確認
- [ ] Premium 時非表示確認

## UX
- [ ] 起動直後に課金ポップアップを出していない
- [ ] 執筆中に広告を出していない
- [ ] 保存直後に広告を出していない
- [ ] 広告がメモカードと誤認されない
- [ ] プレミアム導線が煽りすぎていない
