# Phase 11 Copy Alignment Report

## 目的
ホーム上部コピーの左寄り感、日本語改行、囲みコピーの視覚重心を改善した。

---

## 1. 改善前の課題
- 囲みコピーが左寄りに見えた
- 吹き出し/囲み線の印象が残っていた
- 日本語コピーの改行品質をさらに安定させる必要があった

---

## 2. 修正内容
### 視覚重心
- ホーム上部の導入領域を `max-w-[390px]` の中央カラムに整理した
- 主コピーを `margin-inline: auto` の専用紙片コンポーネントへ分離し、左右差のない幅・padding に統一した
- 検索バーまでの距離を少し広げ、上部コピーが単独で静かに中心へ置かれるようにした

### 紙片化
- 吹き出し風の導入カードから、薄い境界線と柔らかい陰影を持つ和紙札風の紙片へ変更した
- 左右非対称な装飾や translate に頼らず、中央基準の背景・影・内側光だけで質感を整えた

### 日本語改行
- 主コピーを 2 行固定の `span` 構造へ寄せ、意味単位での改行を維持した
- 導入説明文と空状態文言も孤立行が出にくい表現へ調整した
- 英語サブコピーは 1 行の補助要素に下げ、日本語の主役感を保った

### 余白
- ロゴ、主コピー紙片、導入説明、検索バーの間隔を黄金比スケール寄りに調整した
- 空状態の補助文を上部コピーと役割が重なりすぎない内容へ変更した

## 4. 確認した画面幅

* 375px: 紙片は中央に収まり、2 行コピーの孤立行なし
* 390px: iPhone 基準幅で紙片と検索バーの距離が自然
* 430px: 行長が伸びすぎず、英語補助コピーも主張しすぎない
* 768px: 上部コピーは中央に留まり、背景月と競合しない
* 1024px: 紙片の重心が崩れず、広幅でも上部導入が過度に横へ伸びない
