# App Store Submission Checklist

## Apple Developer / App Store Connect
- [ ] Apple Developer Program登録済み
- [ ] App Store Connectでアプリ作成
- [ ] Bundle ID作成
- [ ] SKU決定
- [ ] アプリ名確認
- [ ] カテゴリ決定
- [ ] 年齢制限設定
- [ ] 価格設定
- [ ] 配信国設定

## Build
- [ ] iOS化方針決定
- [ ] CapacitorまたはiOSプロジェクト作成
- [ ] Bundle Identifier設定
- [ ] Version / Build Number設定
- [ ] iPhone実機確認
- [ ] TestFlight確認
- [ ] Release build作成
- [ ] Xcode Archive
- [ ] App Store Connectへアップロード

## Core App
- [ ] メモ作成
- [ ] メモ編集
- [ ] メモ削除
- [ ] 自動保存
- [ ] localStorageまたはiOS保存確認
- [ ] 検索
- [ ] お気に入り
- [ ] 日付印
- [ ] 保存アニメーション
- [ ] Premium時広告非表示

## StoreKit / Premium
- [ ] App Store ConnectでIAP作成
- [ ] Product ID確認
- [ ] StoreKit実装
- [ ] 商品情報取得
- [ ] 購入処理
- [ ] 購入復元
- [ ] Sandboxテスト
- [ ] TestFlightテスト
- [ ] mock購入削除または本番では無効化

## Ads
- [ ] 広告SDKを入れるか決定
- [ ] 広告表示位置確認
- [ ] エディタ広告非表示
- [ ] 保存直後広告非表示
- [ ] Premium時広告非表示
- [ ] SDKのPrivacy Manifest確認
- [ ] SDKのデータ収集内容確認

## Privacy
- [ ] Privacy Policy URL用意
- [ ] App Privacy入力
- [ ] 第三者SDKの収集データ確認
- [ ] トラッキング有無確認
- [ ] ATT必要性確認
- [ ] ATT文言確認
- [ ] 子ども向けでないことを確認

## Metadata
- [ ] アプリ説明文
- [ ] サブタイトル
- [ ] キーワード
- [ ] プロモーションテキスト
- [ ] スクリーンショット
- [ ] アイコン
- [ ] サポートURL
- [ ] プライバシーポリシーURL
- [ ] Review Notes

## Final Review
- [ ] 誇大表現なし
- [ ] 医療/治療表現なし
- [ ] 広告が邪魔ではない
- [ ] 課金導線が過剰ではない
- [ ] 残心の世界観が壊れていない
