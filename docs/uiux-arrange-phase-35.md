# Phase 5 Arrange Report

## 目的

MVP の UI/UX を、残心らしい静けさ・余白・現代和の方向へ磨き込み、
体験そのものを「ただのメモ帳」から「書く前後の気配が残るメモ帳」へと引き上げる。

機能は増やさず、装飾は足さず、ただ整える。

---

## 改善した内容

### 1. 背景・地の質感

- **和紙の繊維感**を SVG `feTurbulence` でタイル状に敷いた（強度 0.045 と極薄、文字を妨げない）
- 既存の金・藍の差し色グラデーションは残しつつ強度を微調整
- **円相（ensō）の透かし**を画面右上に固定で一つだけ忍ばせた
  - opacity 0.06（PC では 0.05）
  - `pointer-events: none` でタップを邪魔しない
  - 「装飾ではなく気配」として、画面に「間」をつくる
- 背景は `background-attachment: fixed` でスクロール時にも動かず、紙の上にいる感覚を保つ

### 2. メモカード（NoteCard）

- 刀の刃のような細い縦線を、カード端ではなく **左余白の内側**（`left: gr-3`）へ移し、角丸の中で完結させた
- お気に入りの縦線は **2px の金**、それ以外は **1px の墨/15%** で、静かに区別
- 影を `shadow-paper-soft` に下げ、**hover で 1px だけ浮く**（紙が呼吸するような表現）
- 本文プレビューに `leading-golden` を適用、読みやすさを保ちつつ静けさを残す
- 更新日時を **uppercase + tracking 0.18em** の小さな英字風タイポへ。「日付」を主張させず、余韻にする
- お気に入りの金印は直径 10px に絞り、装飾ではなく **しるし** に

### 3. ヘッダー

- 「残心」を 28 → 30px、字間を 0.18 → 0.22em に。**より静かで大きく**
- 「ZANSHIN NOTES」は 11 → 10px、字間 0.32 → 0.42em。**囁くような副題**へ
- タグラインを `leading-ample (1.85)` に。詩のように間を置く
- ヘッダー直下に **金のグラデーション細線**（透明→金 35%→透明）を一筋。
  装飾ではなく「序」「跋」のような区切り

### 4. 検索バー

- フォーカス時に **藍の薄い輪郭**（`border-indigo/30`）が現れる ── 静かなフィードバック
- 既定の影を `shadow-paper-soft` に。普段はもっと静かに、触れたときだけ紙が起きる

### 5. 新規作成 FAB

- サイズ 55px はそのまま（タップ領域確保）
- 既定で `shadow-paper-hover` の柔らかい影を持ち、hover で **1px 浮く**
- ストローク幅を 1.6 → 1.5 に。「＋」をより細く、控えめに

### 6. エディタ

- タイトルは 24 → 26px、字間 mincho（0.04em）で **本文の前奏**として大きく
- タイトルと本文の間は、墨の罫線ではなく **金のグラデーション余韻**へ。情報を区切らず、気配で繋ぐ
- 本文は `leading-ample (1.85)` に。**書くときの行間を広く取り、考えが置ける余白**にする
- プレースホルダの濃度を `/60 → /55` に。書く前から主張しないように
- 上部バーのアイコンボタンを 34 → 40px に拡大。**iPhone でのタップが確実に**
- 削除ボタンは hover 時に `vermilion/5` の薄い朱が滲む。**色に意味を込めつつ静か**に
- 保存表示を **金の呼吸点（breathing dot）+ 明朝の語**に変更
  - 「そっと書き留めています」中は金の点が呼吸（2.8s）
  - 「余韻を保存しました」では静止した金の点
  - 文字色のみのバージョンより**身体的で、それでいて静か**

### 7. 削除確認モーダル

- 背景に `backdrop-blur-[2px]` を追加。**現実が一歩遠のく感覚**
- モーダル本体に `animate-softUp`（360ms / cubic-bezier）を適用。下から静かに上がってくる
- 「手放す」ボタンを少し横長に（px 4 → 5）。**重要な所作には間を持たせる**

### 8. 空状態

- 検索 0 件にも、円相に似た**金の細線**を一筋。「見つからない」を寂しくしない
- 文言の字間と濃度を全体で揃え、**画面全体で同じ呼吸**になるよう調整

### 9. アニメーション / 体験

- `washiFade`（600ms）：画面切替時の入場。瞬きより少し長く、画面が紙として現れる
- `softUp`（360ms cubic-bezier）：モーダル等の浮上
- `breathDot`（2.8s 無限）：保存中の生命感
- **リストカードに 40ms ずつ段差のある fadeIn**（最大 6 個まで）。一斉ではなく、順に置かれていく感覚
- `prefers-reduced-motion` を全域で尊重（呼吸も含めすべて停止）

### 10. アクセシビリティ / 触り心地

- フォーカスリングを 2px 直角 → **1.5px / radius 8px / offset 3px** に。鋭くなく、輪郭は残す
- 入力欄（input / textarea）はフォーカスリングを消し、**親要素側で表現**するよう統一
- 削除確認の安全余白、戻るボタンのタップ領域も合わせて調整

## デザイン判断

### なぜ「金の呼吸点」を保存表示にしたのか

文字だけで「保存しました」を出すと、機能の通知になる。残心は通知のアプリではない。
**金の小さな点が呼吸する**ことで、保存は「身体的な余韻」に変わる。
点は装飾ではなく **金（gold = #C9A646）= MVP 定義済みのお気に入り色** を流用しており、
新しい色や記号は増やしていない。

### なぜ「刀の縦線」をカード端から内側へ動かしたのか

カード左端ぴったりに置くと**線が UI の罫線に見える**。
左 13px の余白の中に置くことで、**カードに飾られた一筋**として、装飾ではなく気配になる。

### なぜ円相を透かしにしたのか

円相そのものをロゴとして既に空状態に使っているため、画面全体に**もう一度同じものを置く**と
和風テーマパークになる。だから **opacity 0.06** で、見ようとしないと見えない位置に固定した。
「気づくか気づかないかの境界」が残心らしい。

### なぜ行間を広げたのか（1.618 → 1.85）

書くアプリで最も大切なのは「**書いている最中の余白**」。
行間が詰まっていると、考えが押し出される感覚になる。
1.85（≒√3.4）まで広げることで、句読点と句読点の間にも**間**が生まれる。

### なぜ動きを全部「短く」ではなく「長く」したのか

300ms 以上の transition は一般的に「遅い」とされるが、残心は急がない。
急がせない動きこそが、書く前の心を落ち着かせる。
`prefers-reduced-motion` ユーザーには完全停止を返すので、好みは両立できる。

### 他に行ったこと

- shadow を 3 段階（soft / paper / hover）に整理し、**触ったときだけ紙が起きる**段差を作った
- `tracking.mincho`（0.04em）を導入。明朝体の見出しが詰まりすぎないよう、グローバルに統一
- iPhone のセーフエリア余白に加え、FAB と被らないよう一覧の bottom padding を `89px`（gr-7）に確保

## 守ったこと

- 複雑にしすぎない（新しい画面・新しい設定は **0**）
- 余白を守る（むしろ広げた）
- 侘び寂びを意識する（装飾ではなく気配、しるし、余韻）
- iPhone-first（ボタンは 40px / 55px、底のセーフエリア、タップ領域確保）
- 色はパレット内（washi / paper / sumi / ink-muted / gold / indigo / vermilion）のみ
- スペーシングは黄金比スケール（gr-1 ~ gr-7）のみ
- 機能を増やしていない（メモ作成・編集・削除・検索・お気に入りのみ）
- MVP のデータ構造・localStorage 保存方針・ログインなし・同期なし・AI なし・課金なし、すべて**変更なし**

## 自己評価

### 第一印象
**OK** ── 開いた瞬間に和紙の繊維、円相の気配、金の細線、明朝の「残心」が静かに揃う。
「Japanese minimalism / Wabi-sabi」が伝わる構図になっている。

### 書く体験
**OK** ── タイトルが大きく、本文の行間が広く、保存通知は文字ではなく金の呼吸点。
「書いている最中に画面が邪魔をしない」状態になった。

### 読み返す体験
**OK** ── カードは紙のように落ち着き、お気に入りは金の縦線で**読み返したい順**に並ぶ。
日付は uppercase の小さな英字風で、本文を主役にしている。

### 操作性
**OK** ── FAB は 55px、エディタの操作アイコンは 40px、タップ領域は十分。
hover / active / focus すべてに「紙が呼吸する」一貫した触り心地が入った。

### 残心らしさ
**OK** ──

> このアプリは、機能ではなく「書く前後の気配」で記憶に残るメモ帳になっている。

「開く → 言葉を置く → 余韻が残る → また戻ってくる」の動線で、
余韻（金の点、円相の透かし、明朝の余白）が物理的に**残る**設計になった。
**ただのメモ帳ではなく、「残心」と呼べる**。

## 残っている課題

- ダークモード（墨色の地に金）は未対応 ── MVP 範囲外のため Phase 6 以降の検討
- iPad / 大型タブレットでの中央配置（max-w-zanshin = 720px）は機能するが、左右の余白の演出は最低限のまま
- 円相透かしは「金」よりも「墨の細線」だが、画面によっては好み分かれる可能性あり（opacity 0.06 で許容範囲と判断）
