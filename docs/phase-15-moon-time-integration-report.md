# Phase 15 Moon and Time Integration Report

## 目的
背景の円モチーフを、淡い月・時間・余韻の象徴として再設計し、日付印・紙・言葉との関係を統合した。

---

## 1. 改善前の課題

- 背景円がただの円に見えやすかった
- 月と日付印の関係が弱かった
- 月が時間の象徴としてまだ機能していなかった

---

## 2. 月モチーフの再設計

変更内容:
- `src/components/ZanshinMoon.tsx` を追加し、背景月を専用コンポーネントとして分離
- 背景月を `wash / core / grain` の3層に整理し、朧月・月光・紙の繊維感を分けて調整しやすくした
- 右上から少し画面外へはみ出す配置にし、主コピーやCTAより前に出ない位置へ寄せた
- 月の発光感を弱め、薄金・生成り・薄墨中心の色調に寄せた

---

## 3. 月と紙の統合

変更内容:
- 月の本体に淡い紙色と薄い影を重ね、紙の奥に沈む見え方へ調整した
- 月の上にごく薄い grain レイヤーを重ね、背景紙目と喧嘩しにくい質感に寄せた
- 右上の月が広告や Premium 導線まで落ちてこないよう、一覧下部との距離を維持した

---

## 4. 月と日付印の統合

変更内容:
- `src/index.css` に月光・薄金・薄墨の共通トーンを追加し、背景月と小さな月印の世界観を揃えた
- 日付印の月印を背景月に近いグラデーションへ調整し、保存時の反応も同じ色の流れで見えるようにした
- 保存後アニメーションは維持しつつ、背景月と同じ「月光 → 薄金 → 墨色」の印象で落ち着くように整えた

---

## 5. 意味づけ

残心における関係:

```txt
月 = 時間
日付 = 残ったしるし
紙 = 心の余白
言葉 = そこに置かれたもの
```

この関係を `docs/design-system.md` に追加し、今後の UI/UX 判断基準として明文化した。

---

## 6. 画面幅確認

* 375px: 月は右上奥に留まり、主タイトルへの重なりなし。紙片の背後で静かに見える
* 390px: iPhone基準幅で月の存在感は保ちつつ、コピーの可読性を阻害しない
* 430px: 広めの iPhone 幅でも月が大きくなりすぎず、画面が重く見えない
* 768px: 月が小さく逃げず、上部右側で朧月として認識できる
* 1024px: デスクトップ幅でも右上の余白に留まり、一覧下部の導線とは干渉しない
