# Phase 15 Moon and Time Integration Report

## 目的

背景の円モチーフを、淡い月・時間・余韻の象徴として再設計し、日付印・紙・言葉との関係を統合した。

---

## 1. 改善前の課題

- 背景円がただの円に見えやすかった
- 月と日付印の関係が弱かった
- 月が時間の象徴としてまだ機能していなかった

---

## 2. 月モチーフの再設計

変更内容:
- `src/components/ZanshinMoon.tsx` を追加し、月光の halo・本体・やわらかな core を分けて背景月を整理した
- `src/components/AppShell.tsx` で背景円の直書きをやめ、月専用コンポーネントに置き換えた
- `src/index.css` で背景月を、輪郭の強い丸ではなく、紙の奥に沈む朧月として再設計した
- 月の位置を右上奥へ寄せ、iPhone 幅ではコピー本文に重ならず、PC 幅では端に逃げすぎないサイズへ調整した

---

## 3. 月と紙の統合

変更内容:
- 月に紙目のような繊維レイヤーを薄く重ね、背景装飾ではなく紙の奥にある光として見えるようにした
- `paper-ad-slot` と `paper-premium` の面を少し不透明にし、月光が広告や Premium 導線を強めすぎないよう抑えた
- 月の halo は残しつつ、主コピーと紙片の可読性を邪魔しないよう、右上へ逃がした

---

## 4. 月と日付印の統合

変更内容:
- 日付印の wash・月印・文字色を、月光 → 薄金 → 墨色の流れで背景月と同じ世界観に寄せた
- 保存後アニメーションは維持しつつ、薄金の立ち上がりを少し弱めて、背景月と同じ静かな反応に整えた
- 小さな月印が日付より前に出ないよう glow を抑えた

---

## 5. 意味づけ

残心における関係:

```txt
月 = 時間
日付 = 残ったしるし
紙 = 心の余白
言葉 = そこに置かれたもの
```

この関係を `docs/design-system.md` に追記し、今後の UI 判断基準として明文化した。

---

## 6. 画面幅確認

* 375px: 月は右上の外側に寄り、紙片のコピーを邪魔しない。紙片の右端にだけ淡く気配が触れる
* 390px: 主コピー・英字サブコピーとも干渉なし。画面の狭さも強調しない
* 430px: 月の見え量が少し増えるが、CTA やコピーの背後には入らない
* 768px: 月は右上に収まり、紙片・広告・Premium の各面と重ならない
* 1024px: 月の存在感は保ちつつ、画面中央へ出てこない

確認は headless Chrome によるレイアウト計測とスクリーンショットで実施した。
